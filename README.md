# Extract_ (Function. 룬 설정, 자동수락)
Read Only Korean

###########################################################################

이 프로그램을 사용하시기 전 현재 텍스트파일을 읽어주시길 바랍니다..

본 프로그램은 파일, 폴더 등 관리자권한을 필수로 합니다.

관리자 권한으로 꼭 실행시켜주시길 바랍니다.

라이엇 LCU api의 사용으로 정보 처리 속도가 느릴 수 있습니다. 이 경우 10초 내로 정보가 새롭게 바뀌니 재시작 등을 해주세요.

룬 검색은 "챔프이름(라인)" 입력 후 Insert키를 눌러주세요.

또한 룬에 관해 설명드리자면 메인룬 → 서브룬 → 스텟 방향으로 설정됩니다.

따라서 룬 파일의 제목은 "챔프이름.ini"로 해주시고, 내용물은 예를 들어 (쌍따음표 제외)

"메인룬(지배):포식자:돌발 일격:사냥의 증표:끈질긴사냥꾼,서브룬(마법);기민함;물 위를 걷는자,스텟(공격속도)-적응형 능력치-방어력"

이렇게 설정해주시면 감사하겠습니다.

하지만 저렇게 한글로 써서는 안되니 제가 따로 만든 "rune_info.exe"파일을 사용해 거기 나온 코드로 설정해주시면 감사하겠습니다.

즐겜하세요!
